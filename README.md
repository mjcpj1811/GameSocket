Memory Game

Memory Game là trò chơi trí nhớ đối kháng trực tuyến dành cho 2 người chơi.
Mỗi trận đấu gồm 15 vòng, người chơi phải ghi nhớ và nhập lại chuỗi ký tự/số ngẫu nhiên trong thời gian giới hạn.
Người nhập đúng và nhanh hơn sẽ nhận được nhiều điểm hơn.
Trò chơi cho phép đăng nhập tài khoản, thách đấu người chơi khác, và lưu bảng xếp hạng toàn hệ thống.

Dùng TCP/IP và socket.
